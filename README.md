## Problem Statement

<p>Events for blind dating are relatively well known at college festivals and in cafes. Develop a dating app (either web or mobile based is fine) that specifically hosts blind dating events and matches two users based on shared interests, location and other criteria.</p>


## Solution

* Amore – a all-in-one app, for blind dating. Where users can fill basic details, fill up what S/he is up to & Start they dating journey to find the one.
* Features:
* Profile matching based on our matching algorithm.
* One Match at a time & Chatting feature where user can engage
* Features such as — events & hotel booking to engage & find partner.
* Push notification to nearby amore users and partners for Women safety.
